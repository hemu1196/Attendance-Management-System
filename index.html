University Attendance Management System

About the System

This system allows teachers and admins to track student attendance in real-time. Automated alerts for low attendance and absence tracking are built-in.

Teacher Dashboard

Teachers can mark, update, and monitor attendance. It’s secure, fast, and mobile-friendly for classroom use.

Parent Dashboard

A parent dashboard provides parents with real-time insights in their child’s attendance report.

Student Dashboard

Students can view their attendance reports, receive alerts through their dashboard.
